By default Aerobatic will render a friendly error page for HTTP error responses such as `404`, `500`, `403`, etc. The `custom-errors` plugin allows you to override those pages with your own custom pages.

### Usage

```yaml
plugins:
  - name: custom-errors
    options:
      errors:
        500: errors/500.html
        404: errors/404.html
        401: errors/401.html
```
